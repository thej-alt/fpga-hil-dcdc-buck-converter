# Experimental Results

This directory contains the experimental results obtained from the FPGA-based Hardware-in-the-Loop (HIL) implementation of the DC-DC Buck Converter.

## Contents

- **oscilloscope/** – PWM output and converter output voltage measurements.
- **graphs/** – Performance graphs including duty cycle, output voltage, and error analysis.
- **resource_utilization/** – FPGA resource utilization and implementation results.
- **hardware_output/** – Experimental output images captured during hardware validation.

## Summary

The experimental results demonstrate successful implementation of both open-loop and closed-loop control strategies on the ZedBoard FPGA. The measured output closely matches the expected behavior, validating the effectiveness of the proposed HIL system.
