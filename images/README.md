# Project Images

This directory contains the figures used to illustrate the design, implementation, simulation, and hardware setup of the FPGA-based Hardware-in-the-Loop (HIL) DC-DC Buck Converter project.

## Directory Structure

- **architecture/** – System architecture, project block diagrams, PWM concept, and workflow.
- **simulation/** – MATLAB/Simulink models, waveforms, and controller design.
- **vivado/** – Vivado design flow, block design, synthesis, implementation, and FPGA verification.
- **hardware/** – Hardware implementation, ZedBoard connections, and experimental setup.

These images complement the project documentation and provide visual insight into the complete development workflow.
